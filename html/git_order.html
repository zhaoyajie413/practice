    #git_order

    添加：

    git add xx命令可以将xx文件添加到暂存区，如果有很多改动可以通过 git add -A .来一次添加所有改变的文件。

注意 :

    git add -A表示添加所有内容，

    git add . 表示添加新文件和编辑过的文件不包括删除的文件;

    git add -u 表示添加编辑或者删除的文件，不包括新添加的文件。

    推送到github

    $ git push origin master
